# PT Nusantara Heavy Equipment

Website Company Profile untuk perusahaan penyewaan alat berat yang dibangun menggunakan **HTML5, CSS3, dan Vanilla JavaScript** tanpa framework tambahan.

## 📋 Deskripsi

PT Nusantara Heavy Equipment merupakan website company profile yang dirancang untuk menampilkan profil perusahaan, layanan penyewaan alat berat, portofolio proyek, dan informasi kontak guna meningkatkan kredibilitas perusahaan serta menghasilkan prospek pelanggan baru.

Website memiliki desain modern, responsif, cepat, dan SEO-friendly sehingga dapat diakses dengan baik melalui desktop maupun perangkat mobile.

---

## 🎯 Tujuan Website

* Memperkenalkan perusahaan kepada calon pelanggan.
* Menampilkan katalog alat berat yang tersedia.
* Menampilkan pengalaman dan portofolio proyek.
* Meningkatkan kepercayaan pelanggan.
* Mempermudah proses permintaan penawaran harga.
* Mendukung pemasaran digital perusahaan.
